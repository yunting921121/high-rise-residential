High-Rise Residential Precedent Knowledge Base for Gemini Gem
Version: v1.0
Created for: High-Rise Residential Precedent Design Assistant

Purpose
This text file is a compact Gem-readable knowledge base containing 20 high-rise residential precedent cases. It summarizes each case through site context, design issue, concept, strategy, vocabulary, semantic relations, transferable methods, and limitations. Use it together with case_index.json and gem_instructions.md.

Important Rule
Always base answers on this uploaded precedent database. Do not claim that a project has a feature unless it appears in this file or the uploaded full case JSON files. If detailed data is missing, say that it should be verified.

Database Schema
Each case is organized by: Gene_A = site context; Gene_B = design issue / concept / strategy; Gene_C1 = architectural vocabulary; Gene_C2 = semantic relations; Gene_D = resident behavior / feedback / reflection; Transferable Strategies = design moves that can be adapted to new projects.

CASE 01 — Bosco Verticale
------------------------------------------------------------
ID: case_01_bosco_verticale
Aliases: Vertical Forest
Architects: Stefano Boeri Architetti
Location: Milan, Italy
Year: 2014
Building Type: residential towers / vertical forest
Main Themes: vertical forest, planted balconies, urban biodiversity, private outdoor space, environmental facade
Gene_A | Site Context: Located in Milan's Porta Nuova redevelopment context, the project responds to dense urban living and the lack of nature in the city by bringing trees and shrubs onto the residential facade. The towers act as ecological landmarks and private green living environments.
Gene_B | Issue / Concept / Strategy: Issue: high-rise density can disconnect residents from greenery and outdoor life. Concept: a vertical forest where planting becomes part of domestic space and urban biodiversity. Strategy: deep structural balconies support trees, shrubs, irrigation, and maintenance systems.
Gene_C1 | Key Vocabulary: Bosco Verticale, Milan, vertical forest, planted balcony, biodiversity, balcony extension, green facade
Gene_C2 | Key Semantic Relations:
  - Planted Balcony → extends → Apartment Unit
  - Vegetation → filters → Sunlight and Privacy
  - Vertical Forest → contributes to → Urban Biodiversity
  - Balcony Structure → supports → Large Planters
  - Facade Planting → defines → Tower Identity
Transferable Strategies:
  - Use planted balconies as environmental buffers.
  - Design planting as part of structure and maintenance, not decoration.
  - Use greenery to combine privacy, shade, ecology, and identity.
Gene_D | Limitations / Notes: Planting load, irrigation, wind, fire safety, plant maintenance, and long-term facade access must be verified.
Recommended For Questions About: vertical greenery, planted balconies, ecological facade, biodiversity, private outdoor space

## 1. Purpose of Drawing Preprocess

This drawing preprocess translates architectural drawings and diagrams into textual information for Precedent DNA analysis.

The purpose is to extract:

1. **Architectural vocabulary**
   Important spatial elements, stacked blocks, apartment units, cores, courtyards, roof gardens, terraces, sky gardens, and communal amenities.

2. **Spatial relationship descriptions**
   How six-storey blocks, courtyards, apartments, cores, roof gardens, vertical circulation, horizontal stacking, and landscape systems relate to each other.

3. **Overall spatial logic**
   The design logic behind The Interlace as a horizontal high-rise that replaces isolated residential towers with stacked apartment blocks and shared courtyards.

## 2. Drawing Inventory

### 2.1 Typical Floor Plan

The typical floor plan shows a long residential block containing multiple apartment units arranged along a linear slab. The plan includes several vertical cores, stairs, lifts, and service zones distributed along the length of the block.

The block is not a tower floor plate. It is a horizontal residential bar. This is important because the entire project is built from repeated six-storey bars stacked and interlocked with other bars.

The plan also shows different unit sizes and layouts, with living spaces, bedrooms, kitchens, bathrooms, and terraces or balcony-like edges. This allows a variety of apartment types within the long block.

### 2.2 Concept Diagram

The concept diagram shows the transformation from a conventional array of isolated vertical towers into a system of horizontally stacked blocks.

The left side of the diagram shows separate towers standing apart from one another. The right side shows the blocks laid horizontally and stacked to form courtyards, terraces, and roof gardens.

This diagram explains the main typological shift:

**Vertical towers → Horizontal stacked blocks → Interlaced residential village**

### 2.3 Unit Plan Diagrams

The unit diagrams show different apartment types, including simplex, duplex, townhouse, penthouse, and garden-related units.

These unit types indicate that the project is not composed of one repeated apartment only. The stacking system allows different unit relationships to ground gardens, roof gardens, terraces, and superlevel locations.

### 2.4 Exterior Photo

The exterior photo shows how the diagram becomes a built environment. The stacked residential bars are arranged at different heights and angles. Some blocks rest on others, while others bridge across open space. This creates courtyards, gardens, covered spaces, roof terraces, and a strong horizontal identity.

The image confirms that The Interlace is not a tower complex, but a large-scale residential landscape.

## 5. Spatial Relationship Descriptions

### 5.1 Apartment Blocks → stack into → Superlevels

Six-storey apartment blocks are stacked on top of each other to form larger superlevels and a stepped residential topography.

### 5.2 Stacked Blocks → form → Hexagonal Courtyards

The blocks are arranged in hexagonal configurations, creating eight large courtyards.

### 5.3 Courtyards → organize → Residential Community

The courtyards become the main communal spaces that organize social life, landscape, and amenities.

### 5.4 Roof Gardens → occupy → Block Tops

The top surfaces of stacked blocks become roof gardens and outdoor communal or private spaces.

### 5.5 Blocks → bridge over → Open Space

Some blocks span between supports or rest on other blocks, creating covered outdoor spaces and shaded voids.

### 5.6 Apartment Units → occupy → Horizontal Blocks

Residential units are arranged within long horizontal blocks rather than isolated tower floor plates.

### 5.7 Lift Cores → connect → Stacked Blocks

Vertical cores provide access between different stacked levels and apartment blocks.

### 5.8 Landscape Decks → connect → Courtyards and Amenities

Landscape decks, paths, pools, and gardens create continuity between courtyards and community facilities.

### 5.9 Water Bodies → cool → Courtyard Microclimate

Pools and water features can support evaporative cooling and create more comfortable outdoor spaces in the tropical climate.

### 5.10 Stacking Strategy → creates → Private and Communal Terraces

The interlocking massing produces terraces, roof gardens, and outdoor spaces at multiple levels.

### 5.11 Horizontal Blocks → replace → Isolated Towers

The project transforms the common high-rise tower model into a horizontal and interconnected residential system.

### 5.12 Green Network → links → Ground and Roof Landscapes

Landscape is distributed across courtyards, roof decks, terraces, and elevated gardens, creating a multi-level green network.

## 6. Overall Spatial Logic

The Interlace’s spatial logic is based on turning vertical high-rise housing into a horizontal stack of residential bars.

The typological sequence can be summarized as:

**Point Towers → Horizontal Apartment Blocks → Hexagonal Stacking → Courtyards → Vertical Village**

The residential sequence can be summarized as:

**Apartment Unit → Six-Storey Block → Superlevel Stack → Courtyard Community**

The landscape sequence can be summarized as:

**Ground Garden → Courtyard → Roof Garden → Sky Terrace → Green Network**

Through this strategy, The Interlace combines:

1. **High-density residential living**
2. **Large-scale courtyards**
3. **Horizontal connectivity**
4. **Sky gardens and roof terraces**
5. **Tropical climate response**
6. **Diverse apartment types**
7. **Collective residential identity**

The project demonstrates how density can be organized as an interlocking landscape rather than a series of isolated towers.

---

## 7. Design Interpretation

The drawing analysis suggests that The Interlace is a project about reversing the logic of the tower.

The most important design interpretation is:

**The vertical becomes horizontal.**

Instead of stacking units upward in isolated point towers, the project stacks apartment blocks horizontally and interlocks them to create shared courtyards, roof gardens, terraces, and social spaces.

This makes The Interlace a useful precedent for high-rise residential design where the goal is to combine density, community, outdoor space, and landscape in a tropical climate.

The project also shows that high-density housing does not need to be expressed as a field of separate towers. It can become a continuous and interconnected residential fabric.

# Gem Evaluation

This folder documents the evaluation process of the High-Rise Residential Precedent Design Assistant.

The purpose of this evaluation is to examine how the Gem performs when using a curated knowledge base of 20 high-rise residential precedents. The tests focus not only on whether the Gem can retrieve and compare cases, but also on its limitations, including construction details, structural reasoning, regulatory boundaries, image interpretation, and architectural reflection.

## Folder Structure

- `test_prompts.md`  
  Contains the prompts used to test the Gem.

- `raw_conversations/`  
  Contains original conversation records between the user and the Gem.

- `analysis/`  
  Contains analysis and reflections based on the test results.

## Evaluation Focus

The evaluation focuses on the following questions:

1. Can the Gem correctly retrieve relevant precedent cases?
2. Can the Gem transform case knowledge into design strategies?
3. Can the Gem recognize the limits of its knowledge base?
4. Does the Gem hallucinate technical details such as structure, construction, or regulations?
5. How does the Gem interpret images, renderings, and spatial atmosphere?
6. Does a curated 20-case knowledge base limit AI knowledge or create a more focused design assistant?
7. What does this tool suggest for future architectural education and professional practice?
